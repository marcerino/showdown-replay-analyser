# Showdown Replay Analyser
Replays from pokemonshowdown.com can be analyzed using this Browser based tool.
Outputs a Text:
- Player names
- Winner
- Score (Mons Dead/Alive)
- pokemon s Used by Team
- Killattribution (WIP)

Primarily used for Turnaments.
This Browser based:

https://marcerino.github.io/showdown-replay-analiser/
